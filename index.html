Kreiranje aplikacija brzinom Meteora

Šta je Meteor?

Meteor is a complete open source platform for building web and mobile apps in pure JavaScript.

Platform

  • Node.js
  • MongoDB
  • Meteor Blaze (Reactive UI Library)
  • Meteor Tracker (reactive programming)
  • Distributed Data Protocol (REST for websockets)
  • Meteor Livequery (live database queries)

Principi na kojima je zasnovan Meteor

  • Data on the wire
  • One language
  • Database everywhere
  • Latency compensation
  • Full stack reactivity
  • Embrace the Ecosystem
  • Simplicity Equals Productivity

Kako je moguće kreirati aplikacije brže i zašto?

  • Nije potreban serverski api
  • Setup projekta je sveden na jednu komandu
  • Nema potrebe za build procesom
  • Jaka integracija sa 3rd party paketima
